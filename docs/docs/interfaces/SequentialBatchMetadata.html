Interface SequentialBatchMetadata

Complete metadata for sequential batch execution. Includes tier-based pricing information and detailed execution stats.

Tier-Based Pricing Model:

  • Base Quota: 1 per pair (fixed cost)
  • Expansion Quota: count × tier_rate
    • Free: 0.80/expansion (19% discount)
    • Starter: 0.70/expansion (29% discount)
    • Professional: 0.60/expansion (38% discount)
    • Enterprise: 0.40/expansion (58% discount)
